### What I'm building

**[Agent Architecture Designer](https://thejacksoncode.github.io/Agent-Architecture/)** - Visual multi-agent system designer for Claude Code. Drag 28 AI agents onto a canvas, connect them into workflows, run live simulations, and export system prompts. Single HTML file, zero dependencies. Features a unique Five Minds adversarial debate protocol and human-in-the-loop decision gates. 31 versions, 4600 lines of vanilla JS.

**NaukaAI** - AI-powered learning platform with a Socratic AI tutor (Gemini), spaced repetition (FSRS algorithm), gamification system, and learning analytics dashboard. Built with React Native, Expo, TypeScript, and Supabase. 54,000+ lines of code, 97 React components.

**Prompt Engineering Observatory** - 40+ interactive HTML presentations covering advanced prompt engineering techniques, context engineering, agent architecture, multimodal prompts, and security. Built as an educational resource.

**GhostWriter Audio** - Desktop app that reads Claude Code terminal output aloud. Built with Tauri v2 (Rust) + React, supports Microsoft Edge TTS and Google Cloud TTS with 20 customizable themes.

**Craftship AI** - Personal AI atelier with adaptive model selection across 11 LLMs (Claude, GPT-4o, Gemini, o1), 8 budget tiers, and real-time cost monitoring.
